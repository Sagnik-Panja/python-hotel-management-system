# Hotel PMS (Python)

![Python](https://img.shields.io/badge/python-3.10-blue)
![License](https://img.shields.io/badge/license-MIT-green)

A modular CLI Property Management System built with Python and MySQL.

## Features

- Customer lifecycle management
- Room inventory system
- Food ordering module
- Laundry billing
- Unified billing engine
- Persistent MySQL backend

## Architecture

Layered architecture separating UI, logic, and persistence.
